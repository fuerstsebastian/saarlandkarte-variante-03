2. Hosting (GitHub Pages)

Diese Webanwendung wird als statisches Projekt über GitHub Pages bereitgestellt, einem Dienst der GitHub Inc. (88 Colin P. Kelly Jr. St, San Francisco, CA 94107, USA). Beim Aufruf der Seite baut Ihr Browser eine Verbindung zu den Servern von GitHub auf. Dabei kann GitHub die IP-Adresse der Besuchenden zu Sicherheitszwecken verarbeiten und loggen. In den Server-Logfiles werden typischerweise erfasst:

  • IP-Adresse des zugreifenden Endgeräts
  • Datum und Uhrzeit des Abrufs
  • Name und URL der abgerufenen Datei
  • Übertragene Datenmenge
  • Referrer-URL (die zuvor besuchte Seite)
  • Verwendeter Browser und Betriebssystem

Diese Verarbeitung erfolgt auf Grundlage von Art. 6 Abs. 1 lit. f DSGVO zur Gewährleistung der Betriebssicherheit und Stabilität des Webservers. Weitere Informationen finden Sie in den GitHub Privacy Statements.

3. Kartenmaterial (OpenStreetMap / Leaflet)

Diese Anwendung bindet interaktives Kartenmaterial von OpenStreetMap (OSM) ein, um archäologische Fundstellen geografisch darzustellen. OpenStreetMap wird von der OpenStreetMap Foundation (OSMF), St John's Innovation Centre, Cowley Road, Cambridge, CB4 0WS, Großbritannien, bereitgestellt. Beim Laden der Karte fordert Ihr Browser Kartendaten (Kacheln) direkt von den Servern der OSMF an. Dabei wird Ihre IP-Adresse und ggf. weitere technische Browserdaten an die Server der OSMF übertragen. Es werden dabei keine Cookies oder persistenten Tracker durch OSM gesetzt.

Zur Kartendarstellung wird die Open-Source-Bibliothek Leaflet verwendet. Leaflet wird über das CDN unpkg (unpkg.com) geladen. Beim Abruf der Leaflet-Bibliothek von unpkg wird Ihre IP-Adresse an den CDN-Betreiber übertragen. Unpkg wird von npm, Inc. betrieben und stellt lediglich die JavaScript-Bibliotheksdateien bereit.

Weitere Informationen finden Sie in der Datenschutzerklärung der OpenStreetMap Foundation.

Diese Webanwendung bindet Schriftarten von Google Fonts ein, einem Dienst der Google LLC (1600 Amphitheatre Parkway, Mountain View, CA 94043, USA). Konkret werden drei Schriftfamilien verwendet: Inter, Outfit und JetBrains Mono. Beim Laden der Seite baut Ihr Browser eine Verbindung zu den Servern von Google auf, um die Schriftdateien herunterzuladen. Dabei wird Ihre IP-Adresse an Google übertragen. Rechtsgrundlage ist Art. 6 Abs. 1 lit. f DSGVO (berechtigtes Interesse an einer ansprechenden und konsistenten Darstellung). Weitere Informationen finden Sie in der Datenschutzerklärung von Google.

Zur Speicherung Ihrer bevorzugten Einstellungen (z. B. gewählter Farbmodus, Filtereinstellungen) nutzt diese Anwendung den lokalen Speicher Ihres Browsers (LocalStorage). Diese Daten verbleiben ausschließlich auf Ihrem Gerät, werden nicht an Server übertragen und dienen allein dem Nutzungskomfort (Art. 6 Abs. 1 lit. f DSGVO).

6. Keine Tracking-Cookies und Analysetools

Diese Anwendung verzichtet vollständig auf den Einsatz von Tracking-Cookies, Werbe-Pixeln oder Webanalyse-Diensten (wie z. B. Google Analytics). Es werden keine Nutzerprofile erstellt. Es findet kein Nutzer-Tracking statt.
